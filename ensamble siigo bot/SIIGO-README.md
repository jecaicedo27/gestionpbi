# Siigo Automation - Login y Crear Nota de Ensamble

## ✅ Script Final: `siigo-login.js`

### 🚀 Ejecución
```bash
node siigo-login.js
```

### 📝 Flujo Automatizado

1. **Login** → Siigo Nube (siigonube.siigo.com)
2. **Credenciales** → Llena email y contraseña automáticamente
3. **Selección de Empresa** → Click en "Ingresar" de la empresa activa
4. **Dashboard** → Espera carga completa
5. **Menú Crear** → Click en botón verde "+ Crear"
6. **Nota de Ensamble** → Selecciona "Crear Nota de Ensamble"
